
📊 Sales Data Analysis Project

 📌 Overview
This project focuses on analyzing sales data to generate actionable business insights using Microsoft Excel.

The goal was to clean raw data, perform analysis, calculate key performance indicators (KPIs), and present findings through a dashboard.

---

🛠 Tools & Technologies
- Microsoft Excel  
- Power Query  
- Pivot Tables  
- Data Visualization  
- Basic Statistics  

---

 🔍 Data Cleaning
- Removed duplicates  
- Handled missing values  
- Standardized data formats  
- Ensured consistency across all columns  

---

 📈 Data Analysis
- Created Pivot Tables to summarize sales data  
- Analyzed performance by region and product  
- Identified trends and patterns  

---

 📊 KPIs Calculated
- Total Revenue  
- Total Quantity Sold  
- Average Sales  
- Regional Performance  

---

 📉 Dashboard
An interactive dashboard was created to visualize:
- Sales performance  
- Regional comparison  
- Product trends
